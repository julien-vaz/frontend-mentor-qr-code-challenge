### What I learned

I don't know if I did the best way, but I got happy when I was able to place the footer on the bottom of the viewport, while also keeping the card centralized.

```
.attribution {
    position: fixed;
    bottom: 0%;

    padding-bottom: 1rem;    
```

The body was overflowing the viewport because of the margins, so I got happy when I realized that using the DevTools and was able to fix it.
